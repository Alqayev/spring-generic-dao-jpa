This project can easily be build and executed using Maven. Just run:

> mvn clean compile exec:java

This will compile and run the simple command line application.
The application should output the following...

Nottingham (population 126)
 - Harry Chandler is a candlestick maker
 - Tom Butcher is a butcher
 - Dick Baker is a baker
Searching for citizens named Dick...
 - Dick Baker is a baker
Done.
